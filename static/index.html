GO: A URL Redirector

GO removes the diffculty with recalling a website's name. Think of it as a directory listing for sites you and others visit often.

How To

Your favourite weather page might be difficult to remember.
   http://www.cbc.ca/weather/s0000786.html

But by creating a GO link to it you never have to remember that obscure name again. Lets use weather as an example.
   http://go/weather
Now if you browse to http://go/weather you'll end up at your destination. Making it great for sharing with others too.
